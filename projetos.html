Módulo 3: Periféricos (Dispositivos de Entrada e Saída)

Os **Dispositivos de Entrada** enviam dados para dentro do computador (Teclado, Rato, Scanner). Os **Dispositivos de Saída** recebem os dados processados e mostram-nos ao utilizador (Monitor, Impressora, Colunas de Som).

Módulo 4: Estrutura Interna da CPU

O processador possui duas partes fundamentais para realizar as tarefas:
• UC (Unidade de Controlo): Controla o fluxo de dados e organiza as tarefas.
• ULA (Unidade Lógica e Aritmética): Faz os cálculos matemáticos (somas, subtrações) e decisões lógicas.

📝 Exercício Prático 4:

Módulo 5: Organização de Arquivos no Sistema Operativo

Para manter o computador limpo e organizado, utilizamos **Pastas** (Diretórios), que servem para guardar e separar grupos de **Ficheiros** (Documentos, Fotos, Músicas).
